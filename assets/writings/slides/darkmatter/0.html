Greymatter

Created by Noah Grey in 2000

  • free and open source blogging software package
  • the original opensource weblogging software, inspiring WordPress, Blogger, etc.
  • one of the first popularly used blogging platforms on the web
  • features robust productivity options and extensibility, w/ focus on user customization and extensiblity
    • Features main 'Blog' page w/ all recent posts, a calendar with links to archived posts, an archives page with posts organized individually and by month, and a robust search feature.
